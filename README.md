# relying-party
## RUN
```
$ docker-compose build
$ docker-compose up -d
```

## Usage

### Local
1. Add below in hosts file.
```
127.0.0.1 keycloak
```
2. Acccess to ``http://localhost/private`` .

3. Input user name and password.
